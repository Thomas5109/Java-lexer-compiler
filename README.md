# Java-lexer-compiler
Compilador que reproduz o a linguagem Java

Instalação do antlr 4:

    pip install antlr4-python3-runtime 
    pip install antlr4-tools

Junto com o seu analisador.py na mesma pasta:

    antlr4 -Dlanguage=Python3 JavaSubset.g4                   

Para criar os grafico:

    https://graphviz.org/download/
